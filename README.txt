Jog Tracker beta

Возможность авторизации по UUID (получаем токен и сохраняем его в Keychain)

Возможность просматривать свои записи о беге в виде таблицы 

Возможность добавлять и редактировать записи о беге

Возможность добавлять feedback (свайп для открытия левого меню на главном экране -> send feedback). Для того, чтобы убрать клавиатуру на экране создания feedback, нужно сделать свайп вниз

Возможность просматривать еженедельный отчет (общее время, общее расстояние, средняя скорость)
